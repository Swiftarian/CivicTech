# 消防局列管場所 Excel 自動下載小幫手

這個程式可以幫助您自動下載系統中的所有 Excel 檔案。

## 1. 環境準備

請先確認您已經安裝 Python。接著，請在終端機 (Terminal) 執行以下指令來安裝必要的套件：

```bash
pip install -r requirements.txt
```

## 2. 設定網頁元素 (關鍵步驟！)

因為我無法直接看到您的內部系統，您需要幫我填入幾個關鍵的「密碼」(XPath)，程式才知道要點哪裡。

請使用 Chrome 瀏覽器打開您的系統：

1.  **登入網址**：將 `download_script.py` 中的 `LOGIN_URL` 改成您的登入頁面網址。
2.  **維護按鈕**：
    *   **我已經幫您設定好了！** 程式會自動尋找頁面上所有寫著「維護」的按鈕。
    *   您只需要確認網頁上的按鈕文字真的是「維護」這兩個字即可。
3.  **下載按鈕**：
*   **找不到元素**：通常是 XPath 變了，或是網頁載入太慢。可以試著把 `time.sleep()` 的時間調長一點。
*   **下載失敗**：請確認 `downloads` 資料夾是否有被建立。
